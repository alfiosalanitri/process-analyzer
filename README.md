# process-analyzer
Analyze the ram and cpu usage of a process in real time.

![preview](./preview.png)

## Installation
- `sudo ln -s /path/to/process-analyzer.sh /usr/local/bin/process-analyzer`

## Usage
- `process-analyzer [-n] 5` to refresh every 5 seconds
